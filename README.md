## Linux用? CLI ナウキャスト
* CLIからナウキャストで天気の現在の状況を閲覧するスクリプト。
* 場所は関東地方周辺のみ。
* 画像ビューアに [eog](https://help.gnome.org/users/eog/stable/index.html.ja) を使用。
* 10分ほどの誤差は存在する、という仕様。

### How to use?

```bash
1. $ git clone git@github.com:matt-note/nau.git
2. $ chmod +x nau/nau
3. $ sudo cp nau/nau /usr/local/bin
4. $ nau
```
